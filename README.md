# Governance Factors in ESG Investing — student site (BEAM101, Topic 5)

Public GitHub Pages site of **pre-class** materials for the Summer ESG governance session.
Served at **https://jaeyungkim.github.io/governance-esg/** (set in Settings → Pages).

## What's here (pre-class / no-answers only)
- `index.html` — landing page.
- `governance_preclass.html` + `Governance_ESG_2026_v2_preclass.pdf` — the lecture, **questions only** (answers hidden).
- `case_preclass.html` + `ABIL_Case_2026_preclass.pdf` — the African Bank case, **background + questions only**.
- `reading_list_topic5.html` — the core + further reading list.

## Adding the full materials after class
Copy these from the private master repo into this folder and add a link/card to `index.html`:
- **lecture** — slides `governance_full.html` + `Governance_ESG_2026_v2.pdf`; written reading `governance_textbook.html`
- **case** — slides `case_full.html` + `ABIL_Case_2026.pdf`; written reading `case_textbook.html`

The *written reading* files are the flowing-prose textbook editions — `case_textbook.html` is the
companion to `governance_textbook.html`. They contain the analysis and verdicts, so they stay in the
private repo (queued) until each session has run.

## Never publish here
The MCQ quiz/practice files + answer key, the *during*-class HTML (answers behind a click-to-reveal,
so they sit in the page source), and the Harvard Business School case PDF (licensed; provided to
students through the course, not posted). These live only in the private master repo.
